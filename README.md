# AIMLForHealthcare
AI/ML models and algorithms for various healthcare related problems: e.g. detection of lung cancer from CT scans.
The project aims to develop models for nodule detection and cancer stage prediction using CT scan DICOM files as input. 
Database used: LIDC-IDRI images (open-source) available at [https://www.cancerimagingarchive.net/collection/lidc-idri/](url)
